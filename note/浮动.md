# float 浮动的特点

    1. 会脱离文档流

    2. 文字环绕

    3. 在一行内显示浮动的块级元素

    4. 让行内元素可以设置宽高

    5. 可以用margin来移动,但不能使用margin: 0 auto
        因为脱离了文档流没法自适应

# 如何清除浮动

    1. 直接给父容器设置高度

    2. 在浮动元素的末尾增加一个子容器,并在这个子容器身上清除浮动
                                    clear: both;
        但是会使得代码的可读性变差
        添加许多无意义的标签，结构化较差
    
    3. 使用::after伪元素来清除浮动
        父级标签::after{
            content: '';
            display: block;因为伪元素不属于块 行内块 行内元素,所以要给他定义一个元素
            clear: both;
        }

    4. 使用BFC容器特性来清除浮动的负面效果

    5. 给被影响的box添加clear: both;
